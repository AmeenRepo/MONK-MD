## For Termux/Ssh/Ubuntu
```
apt update
apt upgrade
pkg install git -y
pkg install nodejs -y
git clone your <github link>
cd MONK-MD <YOUR BOT REPOSITORY NAME>
npm install / npm i
npm start

```
## For running
```
npm install
npm start

```
